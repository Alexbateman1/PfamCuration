# PfamCuration

A repository for Pfam protein family database curation tools and workflows.

## Description

This repository contains tools and resources for curating protein families in the Pfam database. Pfam is a comprehensive collection of protein families, each represented by multiple sequence alignments and hidden Markov models (HMMs).

## Features

- Tools for protein family curation
- Sequence alignment workflows
- Quality control and validation scripts
- Documentation and guidelines for curators
